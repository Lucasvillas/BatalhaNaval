# BatalhaNaval
Um jogo de batalha naval que utiliza um sistema distribuido P2P e utiliza do protocolo UDP para comunicação

# Como Rodar (mesma maquina):
Abra 2 terminais Shell <br>
Rode o comando: java Main (sua Porta) <!-- Utilize as portas 5000 e 5001 por conveniencia --> <br>
Ip inimigo: localhost <br>
Porta do inimigo: (porta do inimigo) <br>
